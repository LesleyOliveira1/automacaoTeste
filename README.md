# 📄 Automação de Mesclagem de PDFs

Este projeto é uma automação simples em Python para **mesclar arquivos PDF em um único documento**.

O projeto foi desenvolvido com auxílio de conteúdos do canal **Hashtag Programação**, como parte do aprendizado em automação com Python.

---

## 🚀 Funcionalidades

- Mescla múltiplos arquivos PDF em um único arquivo final
- Organização simples de arquivos
- Automação do processo manual de junção de PDFs

---

## 🛠️ Tecnologias utilizadas

- Python 🐍
- Biblioteca para manipulação de PDFs (ex: PyPDF2 ou similar)

---

## 📁 Estrutura do projeto

Projeto/
│
├── Arquivos/ # PDFs de entrada
├── PDF pdfFinal.pdf # PDF gerado após a junção
├── Projeto1.py # Script principal
└── .gitignore # Arquivos ignorados pelo Git


---

## ▶️ Como executar

1. Clone este repositório:
```bash
git clone https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git

2 Acesse a pasta do projeto:

cd NOME_DO_REPOSITORIO

3 Instale as dependências (se necessário):

pip install PyPDF2

4 Execute o script:

python Projeto1.py

## Observações 
Certifique-se de que os arquivos PDF estão dentro da pasta Arquivos/
O PDF final será gerado automaticamente como pdfFinal.pdf
Recomenda-se usar ambiente virtual (.venv) para organizar dependências
